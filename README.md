# CodeConnect: Campus P2P Tech Tutoring App

## 🌍 Sustainable Development Goal (SDG) Focus
**SDG 4: Quality Education** 
CodeConnect supports SDG 4 by improving learning access and skill development for university students. It provides a peer-to-peer platform where students can find targeted, one-on-one academic support for specific programming and tech-related challenges.

## 💡 Project Description
**The Problem:** Computer Science students often get stuck on specific coding bugs (e.g., C++ memory leaks, Python scraping errors). Lecturers have limited availability, and generic online forums lack specific, context-aware guidance. 
**Our Solution:** CodeConnect is a peer-to-peer (P2P) tutoring platform that connects students facing technical roadblocks with tech-savvy peers (tutors) on campus. It allows students to book hourly sessions for targeted guidance, fostering a collaborative learning environment and accessible education for everyone.

## 🔄 Overview of Our SDLC Work
We followed an Agile Software Development process using Scrum methodologies.Our work is organized into the following phases:
* **Planning & Requirements:** Defined the product vision, created user stories, and developed the UML Use Case Diagram to establish system boundaries and actors.
* **Design:** Created low-fidelity wireframes and a high-fidelity clickable prototype focusing on core user flows (Login, Tutor Search, Booking).
* **Testing & Evaluation:** Conducted usability testing on our early prototype, gathered feedback, and iteratively improved the UI/UX.
* **Final Presentation:** Compiled all SDLC artifacts, Agile (Trello) evidence, and prototype iterations into the final project report.
